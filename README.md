# Phoenix Drone Repair Website

Website for a drone repair company that allows users to contact engineers, order spare parts, and request repairs.

## 🌐 Live Demo
[Phoenix Drone Repair Website](https://dronerevive.netlify.app)

## ✨ Features
- Drone repair services
- Spare parts ordering
- Online contact with engineers

## 🛠 Technologies
- HTML
- CSS
- JavaScript
- Bootstrap

## Libraries

- **AOS** — used for scroll animations in testimonials and hero section
- **Swiper** — used for testimonials slider
